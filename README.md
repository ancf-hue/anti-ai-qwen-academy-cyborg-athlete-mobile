# 🏛️ Anti-AI Qwen Academy: Киборг-Атлет Mobile

**Qwen-учитель учит людей становиться Киборг-Атлетами.**

📱 Полностью офлайн-тренажёр рефлексов, тактического мышления и самоконтроля для мобильных геймеров. 100% бесплатно, без рекламы и внешних зависимостей.

---

## 🎮 8 Протоколов тренировки:
1. 🔫 **Свой / Чужой** → Реакция на целеуказание
2. ⚡ **Клатч** → Тактические решения за 2 секунды
3. 🧘 **БОС-Дыхание** → Контроль стресса (4-7-8)
4. 🏛️ **Наводчик** → Гашение отдачи и трекинг
5. 👁️ **Пикинг** → Дисциплина угла (<0.4с пик)
6. 🎯 **Прицел PRO** → Удержание цели при хаотичном движении
7. 🔥 **Флик** → Переключение целей + контроль дружественного огня
8. 🔊 **Аудио** → 3D-локализация звука (требуются наушники)

---

## 🚀 Как запустить:
- 🌐 **Онлайн:** `https://ваш-никнейм.github.io/anti-ai-qwen-academy-cyborg-athlete-mobile/`
- 📥 **Офлайн:** Скачайте ZIP → распакуйте → откройте `index.html` в браузере
- 📱 **На телефоне:** Добавьте ярлык на главный экран через «Поделиться» → «На экран Домой»

---

## 📦 Особенности:
- ✅ 100% офлайн (нет CDN, трекеров, аналитики)
- 🌐 Двуязычный интерфейс (RU / EN)
- 📱 Адаптация под iOS Safari и Android Chrome
